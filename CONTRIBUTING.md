Thanks for your interest. This is a small project but contributions are welcome.

## How to contribute

1. Fork the repo
2. Create a branch (`git checkout -b fix/your-fix`)
3. Make your changes
4. Test on macOS with iTerm2 (or note which terminal you tested with)
5. Open a PR with a short description of what you changed and why

## What I'd love help with

- Terminal.app support (I only use iTerm2, so this needs someone who uses Terminal daily)
- Linux port (the core logic is macOS-specific right now — AppleScript + System Events)
- Better handling of edge cases (multiple terminal windows, split panes)

## What probably doesn't need changing

- The focus-detection approach works well — no need to overhaul it
- Keep it simple. This is a lightweight utility, not a framework.

## Questions?

Open an issue or reach out. I'm responsive.
